####e_ASTROGAM_Analysis   

RELEASE: Version 1.0.3   16/02/2017

AUTHORS:  G. Giannella  &  S. Guidotti

DESCRIPTION: software which analyzes the fits files generated by BoGEMMS, based on Geant4 10.4. This code is tested for eASTROGAM V1.0 version. It runs on Python 2.7 and it needs the modules: astropy, numpy, math, copy, time, sys.

GENERAL USAGE NOTES: 
————————————————————

- To run this code, the path of the fits files which are going to be analyzed must follow the filepath of the simulation. 

- The software needs the two files generated by the eASTROGAM_Geometryv1.py, which must run first.

- The input parameters must be set up using the file eASTROGAM_ANALYSISv1_input.ll

- The eASTROGAM_ANALYSISv1_file_remote.py must be in the same directory of function_file.py
